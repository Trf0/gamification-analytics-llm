# 🎯 Gamification Analytics with LLM

Sistema para transformar relatórios de plataformas de gamificação em insights educacionais usando Python + LLM + Power BI.

## 🚀 O que esse projeto faz?

1. Lê relatórios exportados (ex: Kahoot)
2. Centraliza os dados em uma planilha principal (Google Sheets)
3. Usa um LLM para classificar habilidades dos alunos
4. Gera uma tabela final de alunos vs habilidades
5. Visualiza tudo no Power BI
